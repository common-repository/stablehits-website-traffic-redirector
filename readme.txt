Redirect traffic from StableHits to a random post on your website.

== Description ==

This plugin will automatically redirect traffic sent from [StableHits](https://stablehits.com/) to random posts on your website.

When placing an order at StableHits.com, enter http://yourdomain.com/stablehits/ as the URL to send traffic to.  Remember to replace "yourdomain.com" with your actual domain or the location of your WordPress installation.

WordPress users can enter the coupon code "WORDPRESS" to receive a 30-day trial.  The code will work on any traffic plans that offer up to 10,000 hits per day.  The code will not work on plans that exceed 10,000 hits per day.

== Installation ==

=== From within WordPress ===
1. Visit 'Plugins > Add New'
2. Search for 'StableHits'
3. Activate StableHits from your Plugins page.

=== Manually ===
1. Upload the `stablehits` folder to the `/wp-content/plugins/` directory
2. Activate the StableHits plugin through the 'Plugins' menu in WordPress
